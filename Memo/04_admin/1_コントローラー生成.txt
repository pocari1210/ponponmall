
◆下記コマンドでコントローラー生成◆
php artisan make:controller Admin/OwnersController --resource

コントローラーのファイルが新規作成される
\app\Http\Controllers\Admin\OwnersController.php

◆routeの編集◆
\routes\admin.php

// ログインしている時見れるようmiddlewareをつける
Route::resource('owners', OwnerController::class)
->middleware('auth:admin');

◆コントローラー編集◆
\app\Http\Controllers\Admin\OwnersController.php

コントローラのコンストラクター内でミドルウェアを指定
認証をしていることを確認出来たら各メソッドを
実行できるようになる

auth:adminでguard設定を行う

  public function __construct()
  {
     $this->middleware('auth:admin');
  }
