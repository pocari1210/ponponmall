
QueryBuilder

コントローラー内でDBファザードを読み込み、
テーブルを指定し、データを取得する

・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・

use Illuminate\Support\Facades\DB; // QueryBuilder クエリビルダ

 public function index()
 {
    $q_get = DB::table('owners')
　　->select('name','created_at')
    ->get();
 }

DB::table('owners')
⇒データを取得するテーブル名を指定

select('name','created_at')
⇒selectでnameとcreated_atを指定

get();
⇒全ての情報を取得する
